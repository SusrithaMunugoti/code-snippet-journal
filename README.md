# Code Snippet Journal

A Chrome browser extension that lets you save, tag, and annotate code snippets you find while browsing the web.

## Features
- Highlight any code on a webpage, right-click, and save it instantly
- View all your saved snippets in the extension popup
- Add tags and notes to each snippet
- Search snippets by keyword or tag
- All snippets are stored locally in your browser

## How to Install
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable Developer Mode (toggle in the top right)
4. Click "Load unpacked" and select this folder
5. The extension icon will appear in your toolbar

## How to Use
1. Highlight any code on a webpage
2. Right-click and select "Save to Code Snippet Journal"
3. Click the extension icon to view, tag, and annotate your snippets
